# Snake Ultimate 🐍

**Snake Ultimate** — это современное переосмысление классической игры «Змейка», написанное на TypeScript с использованием Canvas API. Игра обладает богатым визуальным оформлением в стиле ретро, процедурным звуком и множеством настраиваемых механик.

## 🌟 Особенности

- **Полная кастомизация:** Выбирайте размер и форму поля (Прямоугольник, Круг, Ромб, Крест, Случайная форма), скорость игры и количество препятствий.
- **Бесконечное поле:** Поддержка режима игры без границ — выходя за край, змейка появляется с противоположной стороны (работает даже на сложных формах полей!).
- **Разнообразие фруктов:**
  - 🍎 **Яблоко:** +1 к длине, +10 очков.
  - 🍒 **Вишня:** +2 к длине, +10 очков.
  - 🍌 **Банан:** +3 к длине, +10 очков.
  - 🍉 **Арбуз (2x2):** Огромный фрукт. Появляется ненадолго (на 8 секунд, мигает перед исчезновением). Дает временное ускорение (на 10 сек) и +50 очков.
  - 🌟 **Золотое яблоко:** Дает +10 к длине и +100 очков (исчезает через 8 секунд).
  - 🍆 **Баклажан:** Штраф -50 очков и уменьшение длины на 2 блока.
- **Объекты и препятствия:**
  - 🪵 **Пеньки:** При столкновении ломаются, но отнимают 1 блок длины хвоста и 20 очков.
  - 🪨 **Камни:** Неразрушимые препятствия. Столкновение смертельно.
  - 🌀 **Порталы:** Парные телепорты, позволяющие мгновенно переместиться в другую часть карты.
- **Эпичные эффекты:** Частицы при поедании фруктов (в цвет фрукта) и драматичный цепной взрыв змейки при поражении с эффектом тряски экрана.
- **Процедурное аудио:** Уникальные 8-битные звуки для каждого события, сгенерированные с помощью Web Audio API (без внешних аудиофайлов!).
- **Modern UI:** Футуристичный Glassmorphism-интерфейс, совмещенный с классическим пиксельным шрифтом.

## 🎮 Управление

- **Движение:** Стрелки клавиатуры или `W` `A` `S` `D`. На мобильных устройствах — свайпы.
- **Пауза:** Клавиша `P` или кнопка в интерфейсе.
- **Рестарт:** Клавиша `R`.
- **Навигация в меню:** Клавиши-стрелки и `Enter` для быстрого старта.

## 🚀 Запуск проекта

Проект не требует внешнего бэкенда и работает полностью в браузере.

### Локальный запуск (Development)

Убедитесь, что у вас установлен [Node.js](https://nodejs.org/).

1. Клонируйте репозиторий:
   ```bash
   git clone <url-вашего-репозитория>
   cd game
   ```
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите сервер разработки:
   ```bash
   npm run dev
   ```
   Откройте ссылку (обычно `http://localhost:5173`) в вашем браузере.

### Сборка для продакшена (Production)

Для создания оптимизированной сборки (например, для публикации на GitHub Pages):

```bash
npm run build
```
Готовые файлы появятся в папке `dist/`.

## 🛠 Технический стек

- **Язык:** TypeScript / HTML5 / CSS3
- **Рендеринг:** Canvas 2D API (с поддержкой интерполяции кадров для плавного движения)
- **Сборщик:** Vite + Rolldown
- **Хранение данных:** LocalStorage (для рекордов и настроек)

## 📁 Структура проекта

- `src/engine/` — Ядро игры (Игровой цикл, Рендерер, Аудио, AssetLoader, InputManager).
- `src/entities/` — Игровые объекты (логика Змейки).
- `src/game/` — Управление правилами, стейт-машина (GameManager, MapManager).
- `src/ui/` — Интерфейс (Главное меню, Настройки, HUD, Пауза).
- `src/assets/` — Спрайты и текстуры.
